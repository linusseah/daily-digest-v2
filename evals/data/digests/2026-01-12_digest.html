🚀 Hardware Gets Serious

Taalas serves Llama 3.1 8B at 17,000 tokens/second
Canadian startup Taalas announced custom ASIC hardware that serves Llama 3.1 8B at a staggering 17,000 tokens/second—roughly 10x faster than GPU-based inference. 🤖 The Agent Orchestration Layer Emerges

Andrej Karpathy introduces "Claws"
Karpathy tweeted about "Claws"—a new abstraction layer on top of LLM agents for orchestration, scheduling, context management, and tool interactions. He's "a bit sus'd" to run OpenClaw (the viral 200k+ GitHub star implementation) but loves the concept. The term signals a shift from raw LLM → agents → now agent orchestration frameworks. OpenClaw's creator Peter Steinberger recently joined OpenAI while keeping the project open-source, reflecting the tension between security concerns and the push for personal AI agents.
Cursor ships agent sandboxing to make local coding agents safe
Cursor released their "agent sandboxing" system (v2.5, Feb 17) that lets coding agents run freely inside a constrained environment. Agents only prompt for approval when accessing resources outside the sandbox (like network calls), reducing interruptions by 40%. They use Seatbelt on macOS and Landlock on Linux, with granular network access controls. This addresses "approval fatigue" and makes agentic coding workflows practical for production use.
Prompt caching makes Claude Code feasible
Anthropic's Thariq Shihipar explained how prompt caching enables long-running agentic products like Claude Code. By reusing computation from previous roundtrips, they significantly decrease latency and cost. The entire Claude Code harness is built around maximizing prompt cache hit rates, with alerts on cache performance. High cache hit rates enable more generous rate limits—a key unlock for agentic UX.
