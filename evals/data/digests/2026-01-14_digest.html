🛠️ Infrastructure Consolidation

Georgi Gerganov and the ggml.ai team (creators of llama.cpp, the tool that made local LLMs possible in March 2023) have joined Hugging Face. This brings together model hosting, definition, and local inference in one ecosystem. llama.cpp powers Ollama, LM Studio, and countless other local AI tools—it has 95,400 GitHub stars. The project stays open-source and independent, but now has Hugging Face's resources behind it. This is huge for the local AI movement.
How Claude Code Uses Prompt Caching
Thariq Shihipar from Anthropic shared how Claude Code is "built entirely around prompt caching."
