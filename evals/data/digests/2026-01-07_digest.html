🛠️ AI Agents & Developer Tools

Andrej Karpathy on "Claws": The new layer on top of AI agents
Karpathy tweeted about buying a Mac Mini to experiment with "Claws"—systems that give AI agents direct computer control. He frames it as "a new layer on top of LLMs" just like agents were a layer on top of base models. He's cautious about OpenClaw specifically, but clearly believes this is where things are headed. Worth watching closely.
How Claude Code uses prompt caching to enable long-running agents
Anthropic engineer Thariq Shihipar explained how Claude Code is built entirely around prompt caching. By reusing computation from previous roundtrips, they significantly reduce both latency and cost—making long agentic sessions actually affordable. High cache hit rates directly enable more generous rate limits. This is becoming a best practice for any agent harness.
Cursor ships agent sandboxing for safer local coding agents
TLDR AI reports that Cursor described an "agent sandboxing" system that lets local coding agents run freely inside constrained environments, only asking for approval when needed. This addresses one of the biggest concerns with autonomous agents—giving them enough freedom to be useful while preventing catastrophic mistakes.
